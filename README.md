### About the Experiment 

This experiment helps learners understand Fibonacci series generation through two approaches: iterative and recursive. The simulation provides an interactive, step-by-step walkthrough of code execution, variable updates, and recursion behavior. Learners can compare both methods in terms of logic, execution flow, and computational efficiency, and observe why iterative computation is generally more scalable while recursion is useful for conceptual understanding.
